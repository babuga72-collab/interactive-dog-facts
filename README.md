# Interactive Dog Facts 🐶

Welcome to **Interactive Dog Facts**, a small project showcasing information about different dog breeds. This repository includes images, breed information, and simple Python scripts to display and search dog facts. It's designed for learning purposes and as a portfolio project.

## Features

- ✅ Contains facts about popular dog breeds  
- ✅ Includes images for visual representation  
- ✅ Python scripts to display all facts or search for a specific breed  
- ✅ Simple HTML page for an interactive web view
